** Direcciones IPv4

Dada una dirección IPv4 y una máscara de subred se pide calcular: la red, la 
dirección de broadcast y la primera y última dirección de un host.

Este ejercicio no se encuentra en Codewars, sino que es externo. Asi que toca 
hacerlos desde cero. Se deja a elección el formato de entrada de los argumentos.
